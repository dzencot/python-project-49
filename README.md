# Brain Games

Simple games

## Requirements

* make
* Python3

## Installation

* Clone this repo
* Install deps
  ```bash
  make install
  ```

## Usage

```python
make brain-games

make brain-even

make brain-calc

make brain-gcd

make brain-progression

make brain-prime
```
